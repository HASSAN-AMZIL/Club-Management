# Football Scouting Web Application with AI Recommendations and Player Reports

## Project Overview

This university Final Year Project is a Django-based football scouting web application.

The system is used to manage football clubs, players, player statistics, recommendations, and scouting reports.

## Main Goals

- Manage football clubs and players.
- Store and organize player statistics.
- Recommend players based on similarity and possible transfer needs.
- Generate automatic scouting reports for players using AI or rule-based logic.

## Core Features

1. Player recommendation system:
   - Similar player recommendations.
   - Transfer recommendation ideas.

2. AI-generated scouting reports:
   - Automatic textual reports for players.
   - Can use a local LLM such as Ollama, Mistral, or Llama 3.
   - Can fall back to simple rule-based report generation.

## Tech Stack

- Django with Python.
- MySQL using XAMPP.
- Simple HTML and CSS frontend.
- Optional local AI model integration.

## Development Rules

- Keep the architecture simple and clean.
- Build at a Final Year Project level, not enterprise level.
- Avoid overengineering.
- Use clear Django models and relationships.
- Focus on correctness and readability.
- Avoid unnecessary tables or complexity.
